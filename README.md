# 🎓 Student Report System

A web-based Student Report System developed using Python Django. The application allows users to manage student records through complete CRUD (Create, Read, Update, Delete) operations with a responsive and user-friendly interface.

🔗 **Live Link:**
https://student-report-system-3xyg.onrender.com

## 📌 Project Description

The Student Report System is designed to simplify the management of student information. Users can add, view, edit, and delete student records efficiently through an intuitive web interface.

---

## ✨ Features

* Add Student Records
* View Student Details
* Update Existing Records
* Delete Student Records
* Filter Students by Minimum Marks
* Responsive User Interface
* Modern Dashboard Design

## ⚙️ CRUD Functionalities

### ➕ Create

Allows users to add new student records.

### 📋 Read

Displays all student information in a structured table.

### ✏️ Update

Enables modification of existing student details.

### 🗑️ Delete

Allows removal of student records with confirmation.

## ▶️ How to Run

```bash
git clone https://github.com/saiharish14/task-2-saiharish.git
cd task-2-saiharish
pip install -r requirements.txt
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
